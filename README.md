# Grokking Formulation

研究项目，探索神经网络中的 Grokking 现象及其理论解释。

This project aims to investigate the Grokking phenomenon in modular arithmetic as a probe for understanding the emergence behaviors observed in contemporary large models, thereby advancing learning paradigms. It is a theoretical research project.

## 背景 / Background

Grokking 是指神经网络在长期训练后突然从记忆模式泛化到理解模式的现象。本项目旨在深入理解这一现象并提供理论解释。

Grokking refers to the phenomenon where neural networks suddenly transition from memorization to generalization after prolonged training. This project aims to deeply understand this phenomenon and provide theoretical explanations.
