# Grokking Formulation

研究项目，探索神经网络中的 Grokking 现象及其理论解释。

## 项目结构

```
Grokking_Formulation/
├── src/                    # 源代码
├── experiments/
│   ├── figures/           # 实验生成的图表
│   └── logs/              # 实验运行日志
├── data/                  # 数据集
├── configs/               # 配置文件
└── writing/
    ├── paper/             # 论文手稿
    └── notes/             # 研究笔记
```

## 背景

Grokking 是指神经网络在长期训练后突然从记忆模式泛化到理解模式的现象。本项目旨在深入理解这一现象并提供理论解释。
